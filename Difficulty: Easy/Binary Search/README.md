<h2><a href="https://www.geeksforgeeks.org/problems/binary-search-1587115620/1">Binary Search</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="user-select: auto;"><p style="user-select: auto;"><span style="font-size: 14pt; user-select: auto;">Given a sorted array <strong style="user-select: auto;">arr[]</strong> and an integer <strong style="user-select: auto;">k</strong>, find the position(0-based indexing) at which k is present in the array using binary search. If k doesn't exist in arr[] return <strong style="user-select: auto;">-1</strong>.&nbsp;</span></p>
<p style="user-select: auto;"><span style="font-size: 14pt; user-select: auto;"><strong style="user-select: auto;">Note:</strong> If multiple occurrences are there, please return the smallest index.</span></p>
<p style="user-select: auto;"><span style="font-size: 14pt; user-select: auto;"><strong style="user-select: auto;">Examples:</strong></span></p>
<pre style="user-select: auto;"><span style="font-size: 14pt; user-select: auto;"><strong style="user-select: auto;">Input: </strong>arr[] = [1, 2, 3, 4, 5], k = 4
<strong style="user-select: auto;">Output:</strong> 3
<strong style="user-select: auto;">Explanation:</strong> 4 appears at index 3.</span></pre>
<pre style="user-select: auto;"><span style="font-size: 14pt; user-select: auto;"><strong style="user-select: auto;">Input: </strong>arr[] = [11, 22, 33, 44, 55], k = 445
<strong style="user-select: auto;">Output:</strong> -1
<strong style="user-select: auto;">Explanation:</strong> 445 is not present.<br style="user-select: auto;"></span></pre>
<pre style="user-select: auto;"><span style="font-size: 14pt; user-select: auto;"><strong style="user-select: auto;">Input: </strong>arr[] = [1, 1, 1, 1, 2], k = 1
<strong style="user-select: auto;">Output:</strong> 0
<strong style="user-select: auto;">Explanation:</strong> 1 appears at index 0.</span></pre>
<p style="user-select: auto;"><span style="font-size: 14pt; user-select: auto;"><strong style="user-select: auto;">Constraints:<br style="user-select: auto;"></strong>1 ≤ arr.size() ≤ 10<sup style="user-select: auto;">5<br style="user-select: auto;"></sup>1 ≤ arr[i] ≤ 10<sup style="user-select: auto;">6<br style="user-select: auto;"></sup>1 ≤ k ≤ 10<sup style="user-select: auto;">6</sup></span></p>
